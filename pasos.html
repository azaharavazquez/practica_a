Pasos para trabajar con git

Primero se abre la consola y en ésta se pone el comando git clone más la url del repositorio. Seguidamente en la carpera que se ha creado en el escritorio se abre la consola desde la carpeta que se ha crado y se pone el comando de git checkout -b más el nombre de la tarea. se hace la tarea se pone el git add . para validar la tarea, se pone get commit para comentar la tarea, seguidamente se hace git push y se pone el resultado que te da la consola, espero que mi compañera valide y yo hago marge
