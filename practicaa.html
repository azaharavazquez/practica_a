Practica git – Azahara/Dani

Git se usa para poder trabajar en grupo sobre un mismo proyecto sin que haya controversias con las versiones del mismo

Principales ventajas de usar git

  • Sirve para que varios desarrolladores trabajen en un mismo proyecto
  • Se pueden ver en tiempo real cómo van trabajando los diferentes componentes del grupo
  • Evita problemas de pérdida de versiones de proyecto
